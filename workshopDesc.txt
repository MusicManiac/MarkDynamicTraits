[h1]If you're a user[/h1]
Running some mod that makes traits dynamic? Don't know which traits can be earned/lost midgame? At character creation want to pick only static traits? Now achievable (assuming your favorite trait mod added support for this, which is very simple).

[h1]Features[/h1]
[list]
[*] Marks dynamic traits with [~].
[*] That's literally it.

[h1]If you're a modder[/h1]
Very simple to use. Use included .lua file that registers base game traits as guidance.
One nuance that's not in the example file: if you for example do custom UI and you need to display trait names in there without suffix, there are 2 functions available for that so you don't have to bother with it, example use:
[code]
local MarkDynamicTraitsFramework
if getActivatedMods():contains("MarkDynamicTraitsFramework") then
	MarkDynamicTraitsFramework = require("MDTF_Main")
else
	return
end

local characterTrait = CharacterTrait.FIT
local traitDefinition = CharacterTraitDefinition.getCharacterTraitDefinition(characterTrait )
local undecoratedUIName = MarkDynamicTraitsFramework.getUndecoratedUIName(traitDefinition)
local undecoratedUIName2 = MarkDynamicTraitsFramework.getUndecoratedUIName(characterTrait)
local undecoratedLabel = MarkDynamicTraitsFramework.getUndecoratedLabel(traitDefinition)
local undecoratedLabel2= MarkDynamicTraitsFramework.getUndecoratedLabel(characterTrait)
[/code]
Works both with CharacterTrait or CharacterTraitDefinition as you can see.
